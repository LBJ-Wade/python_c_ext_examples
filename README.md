python_c_ext
============

Simple examples for extending Python with C/C++

## Build

Compile the extension module

```bash
$ cd 01_hello_world
$ python3 setup.py build_ext --inplace
```

Run a python script using the extension

```bash
$ python3 ext01.py
```
